# CN / EN 目录映射

本项目采用 CN / EN 双模板结构。CN 面向中文用户，内容目录使用中文；EN 面向英文用户，内容目录使用英文。

工具型目录保持英文稳定，便于 Agent、命令和脚本复用。

## 路径设计原则

1. 内容型目录本地化，保证用户阅读体验。
2. 工具型目录保持英文，保证 Agent 和脚本稳定。
3. 系统规则文件名保持英文，内容本地化。
4. 模板文件名保持英文，内容本地化。
5. CN / EN 结构要尽量保持语义等价，避免一边新增能力另一边长期缺失。
