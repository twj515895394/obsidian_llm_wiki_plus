# 60_Raw_Sources

The evidence layer for raw sources, assets, snapshots, and imports.

| Subdirectory | Purpose |
|---|---|
| `sources/` | Source cards and source records |
| `assets/` | Images, PDFs, audio, video |
| `snapshots/` | Webpage and repository snapshots |
| `imports/` | Imported material from external systems |

Recommended skill: `capture`.
Recommended template: `99_System/templates/source-card.md`.

Raw sources should not be overwritten by AI summaries.
