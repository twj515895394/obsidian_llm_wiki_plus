# 50_Resources

This directory stores tools, links, cases, prompt templates, and other reusable resources.

`50_Resources/` is a resource-list layer. It is not the raw evidence layer and not the structured knowledge layer.

## Recommended subdirectories

```text
50_Resources/
├── tools/
├── links/
├── cases/
├── prompt-templates/
└── raw-resources/
    ├── articles/
    └── papers/
```

## What belongs here

- Tool lists
- Useful links
- Reference cases
- Design materials
- Writing materials
- Prompt templates
- Reference materials that may be useful later but have not yet been integrated into the knowledge base

## Difference from 40_Knowledge_Base

`40_Knowledge_Base/` stores structured, reusable, maintainable knowledge pages such as entities, concepts, workflows, claims, synthesis, and comparisons.

`50_Resources/` stores resource lists and reference materials. Items here are not necessarily processed into knowledge pages yet.

## Difference from 60_Raw_Sources

`60_Raw_Sources/` is the evidence and source archive layer. It stores traceable source cards, assets, snapshots, and imported materials.

If a resource will later support research notes, wiki pages, or decision records, use the `capture` skill to save it into `60_Raw_Sources/` instead of keeping it only here.
