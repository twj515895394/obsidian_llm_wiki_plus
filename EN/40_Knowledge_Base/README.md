# 40_Knowledge_Base

The structured wiki layer for reusable long-term knowledge assets.

| Subdirectory | Purpose |
|---|---|
| `entities/` | People, companies, products, projects, tools |
| `concepts/` | Concepts, terms, theories |
| `workflows/` | SOPs, processes, methods, frameworks |
| `claims/` | Verifiable claims, judgments, conclusions |
| `decisions/` | Decisions that have become reusable knowledge |
| `synthesis/` | Cross-source synthesis |
| `comparisons/` | Comparisons between multiple objects |

Recommended skill: `integrate`.
Recommended template: `99_System/templates/wiki-page.md`.
