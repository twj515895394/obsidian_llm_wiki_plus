# Roadmap

## Current status: v0.7

`obsidian_llm_wiki_plus` now has a publishable first-version template base:

- CN / EN bilingual vault skeletons;
- seven core skills;
- multi-agent entry and command adapter layers;
- system rules, templates, examples, and quality checklists;
- Chinese and English project documentation.

## Candidate work for v0.8

- Add a lightweight structure validation script.
- Add an initialization script for selecting CN / EN and target directory.
- Add more realistic end-to-end examples, such as source → research → integrate.
- Add Obsidian usage suggestions, including recommended plugins, link conventions, and tag conventions.

## Candidate work for v0.9

- Add team usage guidance.
- Add migration guidance for existing knowledge bases.
- Add upgrade guidance between template versions.
- Add more content creation workflow examples.

## v1.0 target

- Stable directory structure.
- Stable core skills.
- Consistent CN / EN documentation.
- Minimal scripted initialization support.
- Release-ready README, examples, and checklists.
