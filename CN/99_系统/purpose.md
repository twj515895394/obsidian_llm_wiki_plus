# purpose.md

## Vault 目标

本 Vault 是一个面向 Obsidian + LLM + Agent Skills 的综合型知识操作系统。

它的目标不是保存更多笔记，而是让资料、研究、知识、项目、内容和决策形成长期可复用的系统。

## 核心使命

```text
外部资料 → 原始资料 → 深度研究 → 知识沉淀 → 项目推进 → 内容创作 → 决策复盘
```

## 适合沉淀的内容

- AI、LLM、Agent、本地模型、工具链等技术研究
- 工作中的系统设计、架构方案、项目经验
- 内容创作素材、选题、文章、视频脚本
- 长期技术选型、产品判断、内容策略
- 高价值 AI 问答与思考过程

## 不追求的目标

- 不追求把所有内容都保存下来。
- 不追求无脑原子化。
- 不追求目录越多越好。
- 不把 AI 总结当作事实来源。
- 不把临时想法直接升级为长期知识。

## 判断标准

一条内容是否值得进入系统，取决于它是否满足至少一项：

1. 后续会复用。
2. 能支撑决策。
3. 能支撑项目推进。
4. 能转化为内容创作素材。
5. 能补充已有知识网络。
6. 能作为某个结论的证据来源。
