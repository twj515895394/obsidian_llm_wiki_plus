# schema.md

## 目录分层

| 层级 | 目录 | 职责 |
|---|---|---|
| 输入层 | `00_收件箱/` | 临时想法、未处理输入 |
| 时间层 | `10_日记/` | 每日计划、记录、复盘 |
| 项目层 | `20_项目/` | 项目推进、交付物、项目复盘 |
| 研究层 | `30_研究/` | 主题研究、分析报告、调研笔记 |
| 问答层 | `35_问答沉淀/` | 高价值 AI 对话与问答结果 |
| 知识层 | `40_知识库/` | 结构化 Wiki，可复用知识 |
| 资源层 | `50_资源/` | 工具、链接、案例、素材 |
| 证据层 | `60_原始资料/` | 原始来源、附件、快照 |
| 输出层 | `70_内容创作/` | 文章、推文、脚本、选题 |
| 决策层 | `80_决策中心/` | 重要判断、选型、策略 |
| 计划层 | `90_计划/` | 待办、审核、路线图、归档 |
| 系统层 | `99_系统/` | 规则、模板、日志、生命周期 |

## 40_知识库 类型

| 类型 | 目录 | 说明 |
|---|---|---|
| entity | `实体/` | 人物、公司、产品、工具、项目 |
| concept | `概念/` | 技术概念、理论、术语 |
| workflow | `方法论/` | SOP、流程、实践方法 |
| claim | `观点/` | 可验证判断、结论、趋势 |
| decision | `决策/` | 决策的知识化摘要 |
| synthesis | `综合分析/` | 跨资料综合理解 |
| comparison | `对比分析/` | 多对象对比 |

## 通用 Frontmatter

推荐知识页使用：

```yaml
---
type: concept
status: active
confidence: 0.7
created: YYYY-MM-DD
updated: YYYY-MM-DD
sources: []
related: []
review_after: YYYY-MM-DD
---
```

## 状态枚举

| 状态 | 含义 |
|---|---|
| draft | 草稿 |
| active | 当前有效 |
| needs-review | 需要复查 |
| stale | 可能过期 |
| deprecated | 已废弃 |
| archived | 已归档 |

## 置信度建议

| 分值 | 含义 |
|---|---|
| 0.3 | 个人初步判断，缺少来源 |
| 0.5 | 有一定依据，但仍需验证 |
| 0.7 | 有较明确来源和分析 |
| 0.9 | 多来源验证或实践验证 |

## 链接规则

- 研究笔记应链接 source card。
- 知识页应链接来源、相关概念、相关项目。
- 决策记录应链接研究笔记、资料和相关知识页。
- 内容创作应链接使用过的研究或知识素材。
