# 99_系统

这里存放 Vault 的系统规则、模板、日志、生命周期和示例。

## 核心文件

| 文件 | 用途 |
|---|---|
| `purpose.md` | Vault 目标和边界 |
| `schema.md` | 目录、知识类型、frontmatter、链接规则 |
| `ingest.md` | 原始资料摄取规则 |
| `review.md` | 待审核机制 |
| `lifecycle.md` | 知识生命周期管理 |

## 子目录

| 子目录 | 用途 |
|---|---|
| `模板/` | 各类产物模板 |
| `日志/` | 摄取、审核、变更日志 |
| `示例/` | 典型产物示例 |
| `归档/` | 系统级归档内容 |
