# 70_内容创作

这里存放面向外部输出的内容草稿、选题、脚本和素材。

## 子目录说明

| 子目录 | 用途 |
|---|---|
| `X/` | X 推文、Thread、Article 草稿 |
| `公众号/` | 公众号文章、专栏草稿 |
| `小红书/` | 小红书图文、标题、封面思路 |
| `视频脚本/` | 短视频、长视频、分镜脚本 |
| `选题库/` | 长期选题、热点选题、栏目选题 |
| `素材库/` | 可复用故事、案例、金句、图片提示词 |
