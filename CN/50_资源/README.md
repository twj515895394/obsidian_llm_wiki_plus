# 50_资源

这里存放工具、链接、案例、提示词模板和其他可复用资源。

`50_资源/` 是资源清单层，不是原始证据层，也不是结构化知识层。

## 推荐子目录

```text
50_资源/
├── 工具/
├── 链接/
├── 案例/
├── 提示词模板/
└── 原始资源/
    ├── 文章/
    └── 论文/
```

## 适合放这里

- 工具清单
- 常用链接
- 参考案例
- 设计素材
- 写作素材
- Prompt / 提示词模板
- 值得后续研究但尚未沉淀的参考资料

## 和 40_知识库的区别

`40_知识库/` 存放已经结构化、可复用、可维护的知识页，例如实体、概念、方法论、观点、综合分析和对比分析。

`50_资源/` 存放资源清单和参考材料。这里的内容不一定已经被加工成知识页。

## 和 60_原始资料的区别

`60_原始资料/` 是证据和来源归档层，用于保存可追溯的 source card、附件、快照和导入材料。

如果某个资源后续会作为研究、知识页或决策记录的依据，应使用 `capture` Skill 保存到 `60_原始资料/`，而不是只放在这里。
