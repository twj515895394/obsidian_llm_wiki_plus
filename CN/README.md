# obsidian_llm_wiki_plus CN Vault

这是中文版 Obsidian Vault 模板。

使用方式：用 Obsidian 打开本目录，并在本目录内使用 Claude Code、Codex、Gemini CLI 等 Agent 工具。

关键入口文件：

- `CLAUDE.md`
- `AGENTS.md`
- `GEMINI.md`
- `.agents/index.md`

核心规则和模板位于 `99_系统/`。

## 建议入口

第一次使用建议先阅读：

- `START_HERE.md`
- `CLAUDE.md`
- `AGENTS.md`
- `.agents/index.md`
- `99_系统/示例/`
