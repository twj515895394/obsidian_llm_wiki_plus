# 40_知识库

这里是结构化 Wiki 层，用于沉淀长期可复用的知识资产。

## 子目录说明

| 子目录 | 用途 |
|---|---|
| `实体/` | 人物、公司、产品、项目、工具 |
| `概念/` | 技术概念、术语、理论 |
| `方法论/` | SOP、流程、实践方法、框架 |
| `观点/` | 可验证判断、主张、结论 |
| `决策/` | 已沉淀为知识的决策结论 |
| `综合分析/` | 跨多个来源形成的综合判断 |
| `对比分析/` | 多个对象之间的比较 |

## 推荐 Skill

- `integrate`
- `research`
- `decision-record`

## 推荐模板

- `99_系统/模板/wiki-page.md`

## 关键要求

知识页应尽量保留来源、更新时间、状态、置信度和待验证问题。
