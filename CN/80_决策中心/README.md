# 80_决策中心

这里记录重要判断和关键决策，方便后续复盘和审计。

## 子目录说明

| 子目录 | 用途 |
|---|---|
| `技术选型/` | 技术方案、工具、框架选择 |
| `架构决策/` | 系统架构、模块边界、工程治理 |
| `产品判断/` | 产品方向、功能取舍、用户价值判断 |
| `内容策略/` | 账号定位、栏目策略、发布策略 |
| `项目路线/` | 项目推进路线、阶段目标、取舍 |

## 推荐 Skill

- `decision-record`

## 推荐模板

- `99_系统/模板/decision-record.md`

## 关键要求

决策记录要说明背景、选项、依据、结论、风险和复查时间。
